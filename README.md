## Quick Start

1. Install dependencies:

```bash
npm install
```

2. Create `.env` in the project root:

```
GEMINI_API_KEY=your_api_key_here
PORT=3000
NODE_ENV=development
```

3. Run the app:

```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser and allow microphone access.
